# JWT_without_signature
Just a short PoC on the risks of not verifying JWT signature in Python.

## Quick Install

```bash
python -m venv venv
source /venv/bin/activate
(venv) git clone https://github.com/vinicius-batistella/JWT_without_signature
(venv) cd JWT_without_signature
(venv) python -m pip install -r requirements.txt
(venv) python app.py
```
